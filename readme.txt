# Professional Portfolio Theme

Custom WordPress portfolio theme built for showcasing development work, projects, and case studies.

## Features

- Custom WordPress theme
- Responsive layouts
- Custom Gutenberg blocks
- SCSS/CSS compilation
- JavaScript bundling and minification
- Optimized asset loading

## Technology Stack

- WordPress
- PHP
- JavaScript
- jQuery
- SCSS/CSS
- Gulp
- Node.js

## Installation

Place the theme inside:

wp-content/themes/

Install dependencies:

npm install

Build assets:

npm run build

Watch files during development:

npm run watch
